Reductions among Problems

Problems

  1. Circuit-SAT: Does a boolean circuit have a satisfying assignment?
  2. CNF-SAT: Does a boolean formula in conjunctive normal form have a satisfying assignment?
  3. 3-CNF-SAT: Does a boolean formula in conjunctive normal form, where each clause has exactly 3 literals, have a satisfying assignment?
  4. Independent Set: Is there a subset of size at least k of vertices of an undirected graph such that no 2 vertices in the subset have an edge between them in the graph?
  5. Vertex Cover: Is there a subset of size at most k of vertices of an undirected graph such that every edge has an endpoint in the subset?
  6. Clique: Does a graph have a complete subgraph of size k?
  7. Set Cover: Given a set U and some of its subsets, are there k subsets whose union is U?
  8. Hamiltonian Cycle: Given an undirected graph on n vertices, does it have a cycle of length n?
  9. Travelling Salesman Problem: Given a weighted complete graph, is there a hamiltonian cycle of weight at most w?
  10. Subset Sum: Given a set S of integers and an integer k, does a subset of S sum to k?
  11. Integer Linear Programming: Solving a linear program where the variables can only take integral values.
  12. Set Partitioning: Given set of numbers, partition it into 2 subsets such that they both have the same sum.
  13. Best Set Partitioning: Given set of numbers, partition it into 2 subsets such that the difference between their sums is at most d.
  14. Hamiltonian Path: Given an undirected graph on n vertices, does it have a simple path of length n-1?
  15. Knapsack: Given n items, each with a certain profit and size, and given a knapsack of limited capacity, can we pack a subset of items to get a profit ≥ P?
  16. Graph 3-coloring: Given a graph, is it possible to color each vertex with one of 3 colors so that no 2 adjacent vertices have the same color?
  17. Directed Hamiltonian Path: Given a directed graph on n vertices, does it have a path of length n-1?
  18. Directed Hamiltonian Cycle: Given a directed graph on n vertices, does it have a cycle of length n?
  19. Makespan scheduling: Given n jobs to run on k processors and the time required for each job, is it possible to finish all jobs in time ≤ T?
  20. Bin Packing: Given the size of n items, can they be packed in k bins of capacity C?
  21. Max cut: Given a graph G = (V, E), find a partitioning (S, V-S) such that E∩(S×(V-S)) is maximized.

Reductions

  1. cnf-sat ≤ circuit-sat: Every CNF formula has an equivalent depth-2 circuit.
  2. circuit-sat ≤ cnf-sat: CLRS ed3, Theorem 34.9, page 1080.
  3. cnf-sat ≤ indset: Algorithm Design, Kleinberg-Tardos, Theorem 8.8, page 460. (For every clause k, create i vertices if there are i variables in the clause. Draw an edge between all vertices of a clause and an edge between complementary vertices across clauses.)
  4. 3-sat ≤ᴸ cnf-sat: A 3-CNF-SAT instance is also a CNF-SAT instance.
  5. cnf-sat ≤ᴸ 3-sat
  6. indset =ᴸ vertex-cover: In a graph (V, E), S is a vertex cover iff V-S is an independent set.
  7. indset =ᴸ clique: S is an independent set in graph G iff S is a clique in the complement of G.
  8. vertex-cover ≤ set-cover: For each vertex v, create the set Sv of all edges incident on v. If some of these sets span the set of edges, then the corresponding vertices form a vertex cover.
  9. ham-cycle ≤ᴸ tsp: Given a graph G, construct a weighted graph G' where c(u, v) = 0 if (u, v) in G and c(u, v) = 1 otherwise. Then G' has a 0-weight tour iff G has a hamiltonian cycle.
  10. 3-sat ≤ subset-sum: CLRS ed3, Theorem 34.15, page 1097.
  11. set-cover ≤ ilp: Represent the set cover instance as a linear program. Each set corresponds to a boolean variable and the requirement of covering each element becomes a linear constraint.
  12. subset-sum ≤ partition: CLRS ed3, Exercise 34.5-5, page 1101. See solutions by Bodnar and Lohr.
  13. ham-cycle ≤ ham-path: CLRS ed3, Exercise 34.5-6, page 1101. See solutions by Bodnar and Lohr.
  14. subset-sum ≤ knapsack: For every number x in the set, create an item of weight x and size x. If the target is t, get a knapsack of size t. A subset sums to t iff we can get profit ≥ t.
  15. partition ≤ subset-sum: Let s be the sum of set X. X can be partitioned into 2 subsets of equal sum iff there is a subset of X of sum s/2.
  16. knapsack ≤ ilp
  17. cnf-sat ≤ ilp: Add a constraint for each clause. For example, '(u1 or !u2 or u3)' becomes 'u1 + (1-u2) + u3 ≥ 1'.
  18. vertex-cover ≤ ham-cycle: CLRS ed3, Theorem 34.13, page 1091.
  19. 3-sat ≤ 3-color: CLRS ed3, Problem 34.3, page 1103. See solutions by Bodnar and Lohr.
  20. 3-sat ≤ d-ham-path: Computational Complexity (internet draft), Arora and Barak, Theorem 2.18, page 52.
  21. 3-sat ≤ d-ham-cycle: Computational Complexity (internet draft), Arora and Barak, Theorem 2.18 (page 52) shows a polytime reduction from 3-sat to directed hamiltonian path. In the directed graph constructed there, add an edge from the end vertex to the start vertex.
  22. partition ≤ makespan: Let X be a set of integers with sum s. For each integer x, create a job which requires x units of time. Then X can be partitioned into 2 equal-sum subsets iff all these jobs can be finished in time ≤ s/2 on 2 processors.
  23. partition ≤ᴸ bin-packing: Let X be a set of integers with sum s. For each integer x, create an item of size x. Then X can be partitioned into 2 equal-sum subsets iff all these items can be packed in 2 bins.
  24. d-ham-cycle ≤ ham-cycle: For every vertex v in directed graph, create vertices v1, v2, v3 in undirected graph. For every edge (u, v) in directed graph, add edge (u3, v1) in undirected graph. Directed graph has ham-cycle iff undirected graph has ham-cycle.
  25. ham-cycle ≤ d-ham-cycle: Replace every undirected edge {u, v} by 2 directed edges (u, v) and (v, u). Directed graph has ham-cycle iff undirected graph has ham-cycle
  26. best-partition ≤ partition: CLRS ed3, Problem 34-2 d, page 1102. See solutions by Bodnar and Lohr.
  27. partition ≤ best-partition: A partition instance is a best-partition instance with difference 0.
  28. indset ≤ max-cut
